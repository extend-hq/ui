# Extend UI

Extend UI is the documentation and registry app for open source document
components created by [Extend](https://www.extend.ai).

It includes React 19 and Tailwind v4 components for PDF viewing, document
uploads, thumbnails, citations, layout blocks, bounding box citations, document
splitting, and e-signature workflows.

GitHub: [extend-hq/ui](https://github.com/extend-hq/ui)
