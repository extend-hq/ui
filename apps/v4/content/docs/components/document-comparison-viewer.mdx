The Compare Viewer helps users understand what changed between two versions or two related files.

## Useful behaviors

- Side-by-side synchronized scrolling.
- Overlay mode for scanned pages and images.
- Difference summary by page, sheet, section, or row.
- Jump navigation for additions, removals, and modified regions.

## Common slots

| Slot | Purpose |
| --- | --- |
| `leftViewer` | Original or baseline document. |
| `rightViewer` | Changed or candidate document. |
| `diffRail` | Change markers and jump navigation. |
