The Metadata Panel gives users the context they need before downloading, approving, or routing a file.

## Useful behaviors

- Normalize metadata across file types.
- Show source, owner, permissions, timestamps, and extraction status.
- Support custom fields from your app domain.
- Distinguish trusted metadata from extracted or inferred metadata.

## Common slots

| Slot | Purpose |
| --- | --- |
| `summary` | Important file facts and status. |
| `fieldGroup` | Related metadata fields. |
| `actions` | Copy, download, open source, or refresh extraction. |
