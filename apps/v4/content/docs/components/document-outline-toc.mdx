The Outline gives every viewer a consistent navigation model even when each file type exposes different structure.

## Useful behaviors

- Heading, bookmark, sheet, and page-anchor navigation.
- Active section tracking as the viewer scrolls.
- Collapsible nested sections.
- Empty and partial-outline states for files with limited structure.

## Common slots

| Slot | Purpose |
| --- | --- |
| `treeItem` | Heading, bookmark, sheet, or page link. |
| `toolbar` | Expand all, collapse all, and filter controls. |
| `emptyState` | No outline available for this document. |
