Search should feel consistent across file types while still respecting each viewer's structure.

## Useful behaviors

- Debounced search with match count and next/previous navigation.
- Highlighted matches in pages, sheets, text files, and OCR overlays.
- Search result grouping by page, sheet, comment, or field.
- Empty, loading, and indexing states.

## Common slots

| Slot | Purpose |
| --- | --- |
| `input` | Query input and keyboard navigation. |
| `results` | Grouped result list with snippets. |
| `highlight` | Inline match rendering inside the active viewer. |
