The Thumbnail Tab Switcher gives users fast spatial navigation through long or multi-page files.

## Useful behaviors

- Virtualized thumbnail tabs for long documents.
- Current tab, loaded tab, selected tab, and error states.
- Drag-to-reorder mode for workflows that edit page order.
- Works horizontally or vertically.

## Common slots

| Slot | Purpose |
| --- | --- |
| `thumbnailTab` | Page preview, index, and state markers. |
| `groupHeader` | Sections, sheets, chapters, or attachment groups. |
| `dropIndicator` | Reorder and insertion feedback. |
