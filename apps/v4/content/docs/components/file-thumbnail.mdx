## Useful behaviors

- Uses `react-pdf` `Thumbnail` for the first page instead of mounting a full PDF page viewer.
- Lazily mounts PDF rendering when the thumbnail approaches the viewport.
- Uses a native `img` tag for image files with browser-level lazy loading and async decoding.
- Shows a shimmering file-preview state while PDF thumbnails render.
- Falls back to a compact file icon when the preview cannot render.

## Common slots

| Slot       | Purpose                                                    |
| ---------- | ---------------------------------------------------------- |
| `preview`  | First-page PDF thumbnail, image preview, or fallback icon. |
| `metadata` | File name, type, size, page count, or processing status.   |
