## Document Previews

The preview can come from an image URL or a rendered preview node. For document thumbnails, the code example above shows:

```bash
npm install react-pdf @extend-ai/react-docx @extend-ai/react-xlsx
```

- Use `react-pdf` to render a first-page PDF thumbnail to a hidden canvas, then pass the resulting object URL to `previewImageUrl`.
- Use `@extend-ai/react-docx` `useDocxViewerThumbnails` to attach a first-page DOCX thumbnail canvas outside the core component.
- Use `@extend-ai/react-xlsx` `useXlsxViewerThumbnails` to paint a sheet thumbnail to a canvas, then pass the data URL to `previewImageUrl`.

## Useful Behaviors

- Keeps the reusable component free of PDF, DOCX, and XLSX renderer dependencies.
- Shows a shimmering file-preview state while an external preview is being generated or loaded.
- Fades the preview in after it loads.
- Falls back to a compact file icon when no preview image is available or image loading fails.
- Supports optional metadata for file name, type, size, page count, or processing status.

## API Reference

### FileThumbnail

The FileThumbnail component renders the reusable thumbnail shell. It accepts externally generated image URLs or preview content without depending on PDF, DOCX, or XLSX renderers.

| Prop               | Type              | Default | Required |
| ------------------ | ----------------- | ------- | -------- |
| `file`             | `ThumbnailFile`   | `-`     | Yes      |
| `className`        | `string`          | `-`     | No       |
| `previewClassName` | `string`          | `-`     | No       |
| `previewContent`   | `React.ReactNode` | `-`     | No       |
| `previewImageUrl`  | `string \| null`  | `-`     | No       |
| `isLoading`        | `boolean`         | `false` | No       |
| `hasError`         | `boolean`         | `false` | No       |
| `showMetadata`     | `boolean`         | `true`  | No       |

### ThumbnailFile

| Prop   | Type     | Default | Required |
| ------ | -------- | ------- | -------- |
| `name` | `string` | `-`     | Yes      |
| `type` | `string` | `-`     | Yes      |
| `size` | `string` | `-`     | No       |
