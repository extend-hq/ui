## Useful behaviors

- Zoom, fit width, fit page, rotate, and print/download actions.
- Virtualized page rendering for long PDFs.
- Optional thumbnail tab switcher, outline panel, search highlights, and annotation layer.
- Text layer support for selection, copy, search, and accessibility.

## Common slots

| Slot      | Purpose                                               |
| --------- | ----------------------------------------------------- |
| `toolbar` | Page navigation, zoom, rotate, and download controls. |
| `sidebar` | Thumbnail tabs, outline, search results, or metadata. |
| `overlay` | Annotations, pins, comments, and OCR alignment.       |
