Citations connect document answers to their source evidence. They should make it easy to jump from an extracted value, comment, or answer back to the PDF page that supports it.

See it composed with the PDF Viewer in the [Citations block](/blocks#citations).

## Useful behaviors

- Page and bounding-box anchors for source evidence.
- Highlighted citation regions that can be focused from outside the viewer.
- Citation lists grouped by answer, field, or reviewer note.
- Empty and unresolved states when the cited page or region is unavailable.

## API Reference

### CitationCard

The CitationCard component renders a clickable citation result that can focus a related PDF page or region.

| Prop              | Type                                            | Default                              | Required |
| ----------------- | ----------------------------------------------- | ------------------------------------ | -------- |
| `label`           | `string`                                        | `-`                                  | Yes      |
| `value`           | `string`                                        | `-`                                  | Yes      |
| `snippet`         | `string`                                        | `-`                                  | No       |
| `pageLabel`       | `string`                                        | `-`                                  | No       |
| `active`          | `boolean`                                       | `false`                              | No       |
| `accentClassName` | `string`                                        | `"border-blue-500/60 bg-blue-500/5"` | No       |
| `className`       | `string`                                        | `-`                                  | No       |
| `...props`        | `React.ButtonHTMLAttributes<HTMLButtonElement>` | `-`                                  | No       |
