The React Excel Editor is an experimental spreadsheet editing surface built on top of `@extend-ai/react-xlsx`. It keeps the viewer shell familiar while enabling direct XLSX workbook mutations through the package editing API.

## Useful behaviors

- Formula bar shows the selected cell's formula, or its display value when no formula is present.
- Font family, font size, font effects, text color, fill color, borders, alignment, wrapping, shrink-to-fit, and number formats use persisted `react-xlsx` cell style APIs.
- Undo, redo, merge, unmerge, sheet add/remove, and export are wired through `react-xlsx` editing APIs.
- Sheet tabs, workbook loading, upload, zoom, table sorting, and dark workbook rendering match the Excel Viewer.
- Large workbooks automatically fall back to read-only behavior above the editor threshold.

## API Reference

### XlsxEditorPreview

The XlsxEditorPreview component renders an editable Excel workbook with upload, theme, zoom, formula editing, cell styling, sheet controls, and export.

| Prop             | Type                        | Default | Required |
| ---------------- | --------------------------- | ------- | -------- |
| `className`      | `string`                    | `-`     | No       |
| `fileName`       | `string`                    | `-`     | No       |
| `isDark`         | `boolean`                   | `-`     | Yes      |
| `onIsDarkChange` | `(isDark: boolean) => void` | `-`     | Yes      |
| `src`            | `string`                    | `-`     | No       |
